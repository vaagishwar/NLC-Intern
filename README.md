# NLC Internship Web Project

## Overview

This repository contains a web application developed during an internship program.  
The project was built as a team effort and focuses on creating a functional, interactive web application using standard web technologies and a Node.js backend.

The goal of this project is to demonstrate practical knowledge of web development, collaboration using GitHub, and deployment-ready project structure.

## Technology Stack

The following technologies and tools were used in this project:

- HTML for structuring the web pages
- CSS for styling and layout
- JavaScript for client-side logic
- Node.js for server-side execution
- npm for dependency management
- SQL (reference schema included)
- Git and GitHub for version control and collaboration

---

### Folder Details

- `public/`  
  Contains all frontend static files such as HTML, CSS, images, and client-side JavaScript.

- `server.js`  
  Entry point for the Node.js server.

- `package.json`  
  Lists project dependencies and scripts.

- `neyveli_eboard.sql`  
  SQL file containing database structure or reference schema.

- `basics.txt`  
  Reference notes related to the project.

## Getting Started

Follow the steps below to run the project locally.

### Prerequisites

Make sure the following are installed on your system:

- Node.js (version 14 or above)
- Git

---

## Clone the Repository

Open a terminal and run:

```bash
git clone https://github.com/vaagishwar/NLC-Intern.git
```
Navigate into the project directory:

cd NLC-Intern

Install Dependencies

Install required npm packages:

npm install

Run the Application

Start the server using one of the following commands:

npm start


or

node server.js


After the server starts, open your browser and go to:

http://localhost:3000

Deployment

This project can be deployed using platforms such as Vercel, Render, or any Node.js-supported hosting service.

General deployment steps:

Push the repository to GitHub

Connect the repository to the hosting platform

Configure build and start commands

Deploy the application
